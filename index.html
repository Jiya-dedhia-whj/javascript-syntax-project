Visual Studio Code is a site where developers of app can develop their apps with more technology and tool.

Visual studio is set like this :


1. Download and install Visual Studio Code.

2. Create a new file in your desktop.

3. Now open Visual Studio Code and click on new file.

4. Now just you need to code.

To open the console in Chrome, use this keyboard shortcut: Cmd + Option + J or Cmd +Shift +J . As an alternative, you could right-click on the webpage and click "Inspect," and the developers window will appear. Click the "Console" tab in that window. You should use the console.
